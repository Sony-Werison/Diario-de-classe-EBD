## Core Features:

- Student Attendance Tracking: Track student attendance (present/absent) for each class.
- Homework Completion: Record whether each student completed their homework assignment.
- Verse Memorization: Track whether students have memorized the verse for the week.
- Behavior Monitoring: Note student behavior during class (good/needs improvement).
- Material Check: Verify if students brought the required material to class.
- Progress Visualization: Display student progress and class statistics via charts.
- AI Behavior Suggestion: Use AI to analyze attendance and homework completion and suggest helpful interventions or recommendations to encourage student behavior.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a sense of trustworthiness and spirituality.
- Background color: Dark slate gray (#1E293B) for a modern, easy-on-the-eyes interface.
- Accent color: Emerald green (#10B981) to highlight positive achievements and milestones.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable, and modern appearance.
